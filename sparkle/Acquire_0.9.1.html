Acquire v0.9.1 Release Notes

Acquire can now play an entire game with Netacquire!

New Features

  • Player is now alerted when it's his turn.

Bugfixes

  • Can't disconnect from a server that has crashed.
  • "Active player" is indicated at odd times (e.g. after the game is over and before the game begins).
  • Network error sometimes comes up immediately or shortly after a display name in use error.
  • Can't send lobby messages.
  • Local games aren't quit properly.
  • Scoreboard columns aren't big enough for two digits.
  • Game log entries don't go to the game log.
  • Phoenix and Quantum are the same colour (purple).
