Acquire v0.9.3 Release Notes

Acquire can now play an entire game with Netacquire!

New Features

  • Better indicated local player's turn in a network game.

Bugfixes

  • Allocate shares sheet no longer shows "for $0".
  • The game board no longer gets focus, which led to annoying keyboarding goofiness.
  • Network game: fixed determination of whose turn it is.
  • Network game: highlighted tiles no longer lose their highlight when the purchase sheet doesn't come up.
  • Network game: joining a game where test mode is already turned on now gives notification of such.
  • Network game: scoreboard now properly updates when players have 0 shares of a hotel.
  • Network game: detecting the end of a game is now done properly.
